# Telegram Film Bot

Бот для поиска фильмов через OMDb API.

## Что делает

- ищет фильмы по названию
- показывает до 3 вариантов
- позволяет выбрать фильм через кнопки
- показывает подробную информацию:
  - название
  - год
  - рейтинг IMDb
  - описание
  - постер

## Технологии

- Python
- aiogram 3
- requests
- OMDb API
- FSM (состояния)

## Как запустить

1. Установить зависимости:
pip install -r requirements.txt


Создать .env файл:
BOT_TOKEN=токен_бота
API_KEY=ключ_OMDb

Запустить бота:
python films_bot.py

Как работает:
пользователь вводит название фильма
бот ищет через OMDb API
показывает список вариантов
пользователь выбирает фильм
бот показывает детали
Автор
